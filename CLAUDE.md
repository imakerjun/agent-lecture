# CLAUDE.md

## 프로젝트 개요

AI 에이전트를 활용한 교육 설계 실험 저장소. 교육 콘텐츠를 에이전트가 생성 → 평가 → 개선하는 워크플로우를 실험한다.

## 디렉터리 구조

```
agent-lecture/
├── CLAUDE.md
├── README.md
├── experiments/              # 실험 기록
│   └── {순번:02d}-{실험명}/
│       ├── design.md         #   실험 설계: 목표, 가설, 입력, 평가 기준
│       ├── runs/             #   에이전트 실행 기록 (덮어쓰기 금지)
│       │   └── run-{순번:02d}.md
│       └── evaluation.md     #   평가 결과, 인사이트
├── rubrics/                  # 평가 기준 (루브릭)
├── agents/                   # 에이전트 시스템 프롬프트
└── inputs/                   # 실험 입력 자료
```

## 실험 원칙

- 모든 에이전트 출력물은 `runs/`에 순번으로 기록한다 (덮어쓰기 금지, 이력 보존)
- 평가 시 `rubrics/`의 기준을 명시적으로 참조한다
- 실험 설계(design.md)를 먼저 작성한 후 실행한다

## 커밋 컨벤션

- 형식: `{동사} "{변경 대상 설명}"`
- 동사: add(신규), update(수정), remove(삭제), fix(오류 수정)
- 커밋 시 `--no-verify` 플래그 사용
